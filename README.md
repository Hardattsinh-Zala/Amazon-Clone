# Amazon-Clone
<br>
Welcome! This is my first project which replicates the basic fuctionalities of Amazon's website. This project is built by using basic HTML and CSS code.
<h2>Features</h2>
<ul>
  <li>Includes navigation bar with different functionalities</li>
  <li>A Hero section </li>
  <li>Shop section with different product categories</li>
  <li>And a Footer with links and panels</li>
</ul>
